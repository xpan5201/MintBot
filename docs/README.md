## 快速开始（uv + .venv）

更完整的安装/启动说明请看仓库根目录 `README.md`。这里给出最短路径：

```bash
uv sync --locked --no-install-project
cp config.yaml.example config.yaml
./.venv/bin/python MintChat.py
```
